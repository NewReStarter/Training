# 题解

### Typical Brutal Enumeration and DFS

对于每个格子而言，翻转偶数次和奇数次等效。因此有：**每个格子存在翻转 0次 和 1次 的情况**
总共16个格子。

最多枚举情况为 C(0,16)+C(1,16)+...+C(16,16) = 2^16 = 65536。 所以可以直接dfs暴搜。

----------


**对于dfs：**
 传入参数为坐标，全局变量cnt计数。

当前个翻则:
    flip(x,y);cnt++;dfs(x+1,y);

枚举不翻则等效翻回来:
    flip(x,y);cnt--;dfs(x+1,y);

为了二维枚举：
    if(x==3){x = 0; ++y;}
